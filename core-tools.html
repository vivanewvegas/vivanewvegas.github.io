IMPORTANT NOTE:

Install all of the following programs to the Tools folder in your Modding Folder unless instructed otherwise

Mod Organizer 2

Installation Instructions:

  • Download the Mod Organizer 2 (Archive) main file
  • Right-click on the downloaded archive and select Properties
  • In the General tab, check the box adjacent to Unblock
  • Select Apply then OK
    • If the option does not show up for you, ignore that step
  • Once you have extracted the archive to your Modding Folder, right-click on ModOrganizer.exe and select Properties
  • Navigate to the Compatibility tab and check the box adjacent to Run this program as an administrator
  • Select Apply then OK

- By far the best mod manager for all Bethesda games, takes advantage of a Virtual File System to keep your Data folder completely clean

FNVEdit

Install normally

- Advanced file viewer/editor and conflict detector. Also known as xEdit
For a good tutorial (shouldn't be necessary if you are just following this guide, but if you want to learn more for yourself), see here

xLODGen

Installation Instructions:

  • Click the download link of the beta version with the highest number (there may only be one link anyways)
  • Extract the archive to your Modding Folder

- Tool for automatically generating LOD, which will be used in the Visuals module. Will mainly be referred to as FNVLODGen

BSA Decompressor

Installation Instructions:

  • Extract the downloaded archive to your modding folder
  • Right-click FNV BSA Decompressor.exe and select Run as administrator
  • Make sure neither Mod Organizer 2 or the game are open, and make sure the path for FNV listed in the program is correct
  • Select Decompress and wait for the program to finish processing
    • If you get an error message about New Vegas being in the Program Files (x86) folder, go back and read Core - Installation to properly install your game
  • Once the program has finished, you can exit it

- Decompresses BSAs and repacks them without zlib compression to improve performance and fixes certain audio files that would not play in vanilla
NOTE: Verifying your game files through either Steam or GOG will undo the Decompressor's effects, so you must rerun it after

IMPORTANT NOTE:

If at any point during gameplay you encounter a lot of missing mesh errors (big red triangles), do the following:

  • Click the button on the toolbar of MO2 and select INI Editor
  • Navigate to the Fallout.ini tab
  • Find the line that starts with SArchiveList=
  • Replace the entire line with the following (needs to all be on one line):
    • SArchiveList=Fallout - Invalidation.bsa, Fallout - Textures.bsa, Fallout - Textures2.bsa, Fallout - Meshes.bsa, Fallout - Meshes2.bsa, Fallout - Voices1.bsa, Fallout - Sound.bsa, Fallout - Misc.bsa
