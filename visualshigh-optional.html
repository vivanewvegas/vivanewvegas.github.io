These are small and simple mods that I didn't think were major enough to be included in the main sections of the guide. They can easily be added on top of any mod setup and don't require any conflict resolution. Inversely, they can also be skipped just as easily without messing anything up:

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Optional Visuals into the pop-up and select OK

Bathroom Doors Fix

Install normally

- Changes all windowed bathroom doors to windowless doors

Strip Wall Billboards

Install normally

- Gives the billboards that surround the Strip actual pictures

High Res Vanilla Posters and Graffiti

Files to install normally:

  • Main File - Hi Res Posters
  • Update File - urbanposters03

- Simple but effective upscale of posters/graffiti

Bighorner Model Variants

Files to install normally:

  • Main File - Bighorner Model Variants - Standard

- Adds a visual difference between Bighorners, their calfs and their bulls

Alsatian NCR Guard Dogs

Install normally

- Changes the NCR guard dogs from feral-looking mongrels to healthy German Shepherds

Fallout 4 Robobrains

Install normally

- Adds an eye and animated bubbling effect to Robobrains, similar to Fallout 4/76

Vanilla Underwear Replacer

Install normally

- Removes the undershirt for males and tweaks the shape of the female top

Better Posture

Install normally

- Stops your character/NPCs from slouching

Cataract Eyes

Install normally

- Fixes the vanilla eyes always looking slightly upwards

Transparent Nuka Bottles

Install normally

- Makes empty Nuka-Cola bottles transparent

McCarran Escalator Glass

Install normally

- Makes the escalator glass in Camp McCarran transparent

McCarran Main Terminal Transparent Glass

Files to install normally:

  • Main File - McCarran_Glass_transparency_NMC_version

- Same thing as the above mod, but for the windows in the main terminal building

Animated Terminal Screen Savers

Files to install normally:

  • Main File - NV Animated Terminals Screen Savers
  • Update File - NV Wall Terminals Animated Screen Savers

- Adds animated screen servers to working terminals

Animated Microfilm Machines

Install normally

- Allows you to activate microfilm machines

Old Word Blues Path Lighting

Hide the following:

Better Pause Menu Screen
