These are small and simple mods that I didn't think were major enough to be included in the main sections of the guide. They can easily be added on top of any mod setup and don't require any conflict resolution. Inversely, they can also be skipped just as easily without messing anything up:

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Optional Content into the pop-up and select OK

Cut Sewers Restored

Install normally

- Makes the Northwest and Southwest sewers accessible and moves the Sealed Sewer back where it belongs

Beyond the Beef Tweaks

Files to install normally:

  • Main File - Beyond the Beef Tweaks

- Improves options and makes the conclusion of the Beyond the Beef quest more fulfilling

I Fought the Law - NCR Start

Files to install normally:

  • Optional File - The Law Won - VNV

- Allows you to start the I Fought the Law quest by picking up a note off of Joe Cobb's corpse and talking to Lt. Hayes in Primm

Securitrons On Alert

Installation instructions

  • Download the 1 of 2 - PLUGINS and 2 of 2 - TEXTURES (256x256 - Vanilla Resolution) main files and start the installation normally through MO2
  • Select the following in the FOMOD:
    • ×

- Colorizes Securitrons portraits and gives them an alternate portrait when they are in combat

Cloud Grenades

Install normally

- Craftable grenades based off the toxic cloud from Dead Money

The Z-43 Hazard Immolation Device

Install normally

- Unique version of the Heavy Incinerator which can be acquired at Big Mountain

Burden to Bear Original

Install normally

- Restores Burden to Bear to its original state, removing the movement speed penalty from medium/heavy armors, instead of just being an exact copy of Strong Back

Immersive Pickup Sounds

Files to install normally:

  • Main File - Immersive Pickup Sounds - Compatibility Version

- Diverse pickup sounds for different types of items
