These are small and simple mods that I didn't think were major enough to be included in the main sections of the guide. They can easily be added on top of any mod setup and don't require any conflict resolution. Inversely, they can also be skipped just as easily without messing anything up:

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Optional Gameplay into the pop-up and select OK

Better NCR Supply Cache

Install normally

- Fleshes out the loot pool of the NCR supply caches obtained through the NCR Radio to make them worthwhile

Honest Hearts Workbench Crate Luck

Install normally

- Randomizes the amount of skill books you receive from a certain container in HH based on your luck skill

Dead Money Toxic Cloud Fix

Install normally

- Changes the damage calculation formula for the toxic cloud in DM to take poison resistance into account

Starting Gear Overhaul

Installation instructions:

  • Download the main file and start the installation normally
  • In the FOMOD, select the following:
    • ×

- Specializes starting gear based on your tagged skills

Vicious Wastes Standalone Mechanics

Files to install normally:

  • Main File - VW Barter Affects Repair Costs
  • Main File - VW Perception Affects Ranged Accuracy

- Makes Perception and Barter more important/viable by giving them more uses

Stat Choices

Install normally

- Allows you to choose from raising HP, AP, or carry weight when leveling up, instead of just automatically boosting HP

NPCs Use Aid Items

Install normally

- Makes NPCs use chems/aid items such as Stimpaks or Med-X
