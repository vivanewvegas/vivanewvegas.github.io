Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Environment + Lighting into the pop-up and select OK

Enhanced Grass

Files to install:

  • Main File - Enhanced Grass No VL

Manual FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to EnhancedGrass_NoVL.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Expand the EnhancedGrass_NoVL.esp header and then the Grass head within it
  • Right-click on the GrassWasteland04 record in the left pane and select Remove
    • Repeat for NVSeaGrass01
  • Select the GrassWasteland05nv record in the left pane
  • Right-click in the right pane and select Hide no conflict and empty rows
  • In the right pane, right-click on the Vertex Lighting box and select Remove
  • Save and exit

- Makes grass and bushes swing more

Interior Lighting Overhaul

FOMOD instructions:

×



FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to Logic and Consistency Fixes.esp and Interior Lighting Overhaul - Ultimate Edition.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input D8052 into the FormID search box in the top left and hit enter
  • Right-click in the left pane and check Hide no conflict and empty rows
  • Drag and drop HELIOS One Control Room option from Logic and Consistency Fixes.esp onto HELIOS One Observation Level from Interior Lighting Overhaul - Ultimate Edition.esp
  • Save and exit

- Modifies all interior cells giving them a realistic look and a more creepy feel
  May cause small performance hits on weaker machines in large interiors such as Camp McCarran

Realistic Wasteland Lighting

Files to install:

  • Main File - FNV Realistic Wasteland Lighting - All DLC
  • Optional File - FNV RWL All DLC - Patch Collection

- Complete weather overhaul designed to make the game look more realistic

A Note About Weather/Lighting Mods:

If you do not like Realistic Wasteland Lighting, the two best alternatives are Clarity and URWLNV. Clarity is a simple tint remover whereas URWLNV is based of RWL and has many different weathers. If you use Clarity, make sure to grab the NV_IFR_CLARITYFIX file on the Interior Fog Remover page

Viva New Vegas - NVGE Preset

Install normally


Special activation instructions:

  • Click the button on the top bar of MO2 and select INI Editor
  • Navigate to the FalloutCustom.ini tab
  • Paste the following into the box:
    • [Water]
    • bUseWaterHiRes=1
    • bReflectExplosions=1
    • bForceHighDetailReflections=1
    • uMaxInteriorWaterReflections=60
    • uMaxExteriorWaterReflections=60
    • bForceHighDetailLandReflections=1
    • iWaterReflectWidth=2048
    • iWaterReflectHeight=2048

- A preset for NVGE designed to make the desert feel warmer and more realistic, including overhauled water
NOTE: If you experience laggy wave effects, restart your PC to clear the caches
The water in NVGE, while beautiful, can be very buggy for some people. If this is the case for you, you can disable it by setting Water to 0 under [Shaders] in the NewVegasReloaded.ini. You can also remove the INI tweaks
