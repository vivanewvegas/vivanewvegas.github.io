These are small and simple mods that I didn't think were major enough to be included in the main sections of the guide, or tools that aren't necessarily needed for just following the guide (but rather making/editing mods). They can easily be added on top of any mod setup and don't require any conflict resolution. Inversely, they can also be skipped just as easily without messing anything up:

Optional Tools

Install all of the following programs to the Tools folder in your Modding Folder unless instructed otherwise:

Paint.net

Install normally

- Photo viewing/editing tool I always use to handle .dds files (the format used for texture files)

BSA Browser

Install normally

- Allows you to browse files within .BSAs and extract them

NifSkope

Install normally

- Tool for basic creating and editing of meshes

Sniff

Install normally

- Streamlined mass nif editor

GECK

Installation instructions:

  • Click the Download the Garden of Eden Creation Kit for Fallout New Vegas here link
  • Extract the contents of the archive to the Root folder

- Tool that allows you to create/edit any data from the game

Mod Organizer 2 Plugins

Installation Instructions:

  • Select the green button in the top left labeled Code and then select Download ZIP
    • ×
  • Extract the following from the archive to Mod Organizer 2/plugins
    • /data
    • pySyncModOrder.py
    • pyMergePluginsHide.py

- Adds two new features to MO2, the ability to hide plugins that have been merged and the ability to sync profile mod orders with one another

Optional Utilities:

GECK Extender

Installation instructions:

  • Download both main files manually
  • Extract the nvse folder from the Geck Extender file to the Data folder
  • Extract the contents of the GECK Patcher archive to the root folder
  • Right-click on Patcher.exe and select Run as administrator
  • In the program, select Patch and then exit once its finished

- Extends GECK functionality and fixes many bugs. Only needed if you installed the GECK above

Console Paste Support

Installation instructions:

  • Download the mod manually
  • Extract the nvse folder from the archive to the game's Data folder

- Allows you to paste text into the console, along with a few other keyboard shortcuts

Discord Rich Presence

Installation instructions:

  • Extract the contents of the archive to the Root folder

- Integrates Discord Rich Presence into Fallout NV

Optional Bug Fix/QOL Mods:

Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Optional Core Mods into the pop-up and select OK

Enslaving Arcade Fix

Install normally

- Allows you to give Arcade to Caesar at any point during the 'Et Tumor, Brute?' quest

Melee Reach Fixed

Files to install normally:

  • Optional File - ALTERNATIVE mastahbossu's Scripted Version

- Fixes the range of melee weapons to better represent their size

Sierra Madre Exterior Fixed

Install normally

- Fixes the messed up texture on the Sierra Madre exterior

Psycho and Slasher Bonus Damage Fix

Install normally

- Fixes the Psycho and Slasher damage buffs not stacking

Clicking Ambient Sound Fix

Install normally

- Fixes faint clicking in some ambient tracks that most people won't hear, but some audiophiles might

No Muzzle Flash Lights

Install normally

- Removes the lights casted on the environment by muzzle flashes, which can be very performance intensive especially on weaker systems

No Snow in Globes

Files to install normally:

  • Main File - No Snow In Globes
  • Optional File - No Snow for Honest Hearts

NOTE: If you do not know what to do when installing multiple files from the same page through MO2, go back to the Core - MO2 Config page and read the IMPORTANT: Mod Installation Advice in Mod Organizer 2 section

- Removes the snow effect in snow globes which can massively boost FPS in the Lucky 38, even on high-end PCs

Motion Sickness Destroyer

Install normally

- Removes all motion blur, double vision, and other nasty nauseating effects that reduce visual clarity. Only install if those effects bother you

Faster Pip-Boy Animation

Files to install normally:

  • Main File - Faster Pip-Boy Animation (2x)

- Doubles the speed of the Pip-Boy opening animation
