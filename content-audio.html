Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Audio into the pop-up and select OK

All Weapon Sounds Overhaul

Install normally

- High quality sound replacers for most guns

All Weapon Sounds Overhaul ESP Replacer

Install normally

- Scripts all the changes from AWSO to avoid conflicts

All Explosion Sounds Overhaul

Files to install normally:

  • Main File - All Explosion Sounds Overhaul BSA and YUP Patch
  • Optional File - All Explosion Sounds Overhaul SCRIPT Version

- High quality sound replacers for all explosions

Passive Combat Music Tweaked

Files to install normally:

  • Main File - New and Improved

- Combat music will only play if you spot an enemy in your crosshair, if you hit an enemy, or if an enemy tries to attack you

Mojave Music Radio Extended

Installation instructions

  • Install all of the main files normally
  • Once the files are installed, hide the following from the MMRE ESPs file:
    • MojaveExtended20.esp
    • MojaveExtended40.esp
    • MojaveExtended60.esp
    • MojaveExtended80.esp
      • Only one .esp is needed for this mod

- Adds 100 new songs to the Mojave Music Radio
NOTE: This is just my personal favorite radio mod, you can use any other radio mod if you want (such as CONELRAD 640-1240)
