Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input Perks/Traits into the pop-up and select OK

Opposite Traits

Files to install normally:

  • Main File - FNV Opposite Traits (YUP OWB)

- 10 new balanced traits with opposite effects of some vanilla traits

Improved Traits

Install normally

- Balances/changes many vanilla traits to make the choice between them more difficult and adds two more traits inspired by classic Fallout

Improved Traits Improved

Install normally

- Fixes/improvements to the Improved Traits mod. See in the included read-me for a list of changes

Baddarkl Perk Overhaul

Install normally

- Overhauls/balances many commonly neglected perks and adds a few new traits/perks inspired by classic Fallout
