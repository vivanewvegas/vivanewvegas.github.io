Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select All Mods -> Create Separator
  • Input LOD into the pop-up and select OK

Installing LOD Assets:

FNVLODGen

Files to install normally:

  • Main File - FNVLODGen Resources

- Resources required for proper LOD generation

LOD Additions and Improvements

Install normally

- New low-poly models for FNVLODGen

Less Horrendous Vikki and Vance Casino LOD Mesh

Install normally

- Replaces the terrible LOD model and textures of the Vikki and Vance

NMC-Beuno LOD Texture Patch

Files to install normally

  • Main File - FNV Poco LOD Texture Patch

- Patches LOD files to use Poco's textures as a base and NMCs textures for anything not covered by Poco

ETL - Terrain LOD Noise

Install normally

- Improves the generic LOD noise texture. Edited to lower the texture sizes to reduce shimmering. Original mod by tgspy

Generating LOD:

NOTE: If you installed all the mods under the Terrain and Major Objects header in the Textures/Models/Animations section, just install the following normally and ignore the instructions below: FNVLODGen Output

The following steps are only needed if didn't install all the mods under the Terrain and Major Objects header:

  • Click the button on the toolbar of MO2 and select INI Editor
  • Navigate to the Fallout.ini tab
  • Find the line that starts with SArchiveList=
  • Replace the entire line with the following (needs to all be on one line):
    • SArchiveList=Fallout - Invalidation.bsa, Fallout - Textures.bsa, Fallout - Textures2.bsa, Fallout - Meshes.bsa, Fallout - Meshes2.bsa, Fallout - Voices1.bsa, Fallout - Sound.bsa, Fallout - Misc.bsa
      • NOTE: This step is only necessary if you used the BSA Decompressor from Core - Tools
  • Exit out of the INI Editor and click the button on the top bar of MO2
  • Select FNVLODGen (if you do not see FNVLODGen listed, go to the Core - MO2 Config section for instructions on how to add it to MO2)
  • In the Arguments box, add -o:"FNVLODGen path\Output"
    • Make sure to change the path to the actual file path for your modding folder, for example -o:"C:\Games\Fallout New Vegas Modding\FNVLODGen\Output" (you will need to manually create the Output folder)
    • Makes sure you keep the -fnv argument as well, separated from the -o argument by a space
  • Select Apply and then exit
  • Run FNVLODGen through MO2
  • In the left pane, right-click and select Select All
  • In the right pane, change both Atlas size options to 8192
  • Check the Terrain LOD option and un-check the Trees LOD option (Tree LOD is non-functional in New Vegas, actual LOD for trees is included in Object LOD)
  • Leave the rest of the settings default (your default settings maybe be different than mine, and some settings will be grayed-out, but you should still leave them be
    • ×
  • Select Generate
  • Let the generation process run (took around 50 mins on my i5 7500) and close the program once the message LOD Generator finished (you can close this application now) appears
  • Navigate to the file path you put in the Arguments box
  • Control+right-click on the meshes and textures folders, add them to a single archive (with 7-zip or WinRar), and install the archive normally through MO2
    • Title the mod FNVLODGen Output
