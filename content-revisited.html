Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Vanilla Quests + Locations Revisited into the pop-up and select OK

Functional Post Game Ending

Files to install:

  • Main File - Functional Post Game Ending
  • Optional File - Functional Post Game Ending - Outside Bets Patch

- Adds a fully functional post-game world to the game that changes based on your choices in-game

Unfriendlier Persuasions

Install normally

- Makes the quest Unfriendly Persuasions more interesting and dynamic by giving the player a sense of tension

Boulder City Showdown

Install normally

- Adds some small tweaks to the Boulder City Showdown quest to make it feel like an actual showdown

Beyond the Beef Tweaks

Install normally

- Improves options and makes the conclusion of the Beyond the Beef quest more fulfilling

Vault 22 Flora Overhaul

Hide the following:

  • vault22FloralOverhaul.esp

- Modifies Vault 22 with more flora, ivy, bushes, grasses, etc.

Vault 22 Flora Overhaul ESP Replacer

Files to install:

  • Main File - 4.1. Content - Vault 22 Flora Overhaul ESP Replacer

- Removes new creatures, weapons, poison effects, crafting recipes, etc. (really any out of place changes), along with a few other misc tweaks.

Decorated BOS Bunker

Install normally

- Adds a many decorations to the otherwise rather empty Brotherhood of Steel bunker
