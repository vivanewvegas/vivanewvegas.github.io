Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Visuals - Finish Line into the pop-up and select OK

Visuals - Conflict Resolution

Files to install:

  • 2.5. Visuals - Conflict Resolution FOMOD

FOMOD Install Instructions:

  • Pick the All in One version if you installed all the mods from this step. If not, choose the Individual Patches option and choose all that apply to you

- Resolves conflicts between the mods from this module

Visuals - Load Order.txt

Files to install:

  • Main File - 2.6. Visuals - Load Order.txt

Special install instructions:

  • Extract loadorder.txt to Mod Organizer 2/profiles/Viva New Vegas - Visuals
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Ensures your load order is correct for the Visuals module

  NOTE: This load order assumes you followed the entire guide with all the recommended options
  For example, if you installed your own mods, those plugin-ins will need to be moved back into the correct spot after the new load order is applied

Using zMerge

  • Verify Visuals - Conflict Resolution.esp is above RWL in your load order
  • Run zEdit through MO2
  • From the top drop-down menu, select zMerge, and from the bottom select Fallout NV
  • Click Start Session
  • Click the Create Merge button
  • In the Details tab, change the Name to Visuals - Merge and make sure the Merge Method is Merge down (improved)
  • In the Plugins tab, select the following:
    • STDG.esp
      Bathroom Doors Fix.esp
      Lucky Shades use Authority Glasses Model.esp
      Boones Machete is a Broah Machete.esp
      Sheriff Meyers.esp
      PipBoysToThePeople.esp
      SimpleGlowingRangerVisors(NeckCover).esp
      Brotherhood of Steel Uniforms.esp
      NVGustsyHandyAnimatedEyes.esp
      NVAnimatedTerninalScreenSaver.esp
      CIB Bighorners.esp
      CIB Bighorners - HH.esp
      Mother Lupa.esp
      Vicious Violetta.esp
      Strip Wall Billboards.esp
      Character Expansions Revised - UPP Addendum Patch.esp
      Character Expansions Revised - UPP Patch.esp
      FP gun follows crosshairs.esp
      OWB-Path Lights.esp
      Burning Campfire - HH Patch.esp
      EnhancedGrass_NoVL.esp
      Oxenloadscreens.esp
      Visuals - Conflict Resolution AIO.esp OR whatever patches you selected from the Conflict Resolution FOMOD
  • Click OK to exit the merge menu
  • Click the Build button in the top right to the right of the merge name
  • The merge should build successfully and you can exit out of the program
  • Right-click in the left pane of MO2 and select All Mods -> Refresh
  • The merge should appear in the left pane and activate it like normal
  • Move Visuals - Merge.esp above Interior Lighting Overhaul - Ultimate Edition.esp
  • Click the on the top bar of MO2 and select Merge Plugins Hide
  • Right-click on the Visuals - Merge listing, select Disable plugins, and then close
    • All the mods that were merged should now appear grayed-out in the left pane of MO2
  • Use the same method to reactivate the mods when you need to rebuild the merge
    • When you reactivate the mods, they will reappear in your load order in alphabetical order. Reinstall the Load Order.txt file to get it back into order

Testing the Game

  • Run the game through MO2
  • Start up a new game and quickly run through the tutorial in Doc Mitchell's house
  • Once you have exited Doc's house, open up the MCM menu to verify all the entries display correctly
  • Spend at least 15-20 minutes in game doing whatever to make sure the game is running smoothly
