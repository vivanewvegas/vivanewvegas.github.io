Overhauls

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Overhauls.
This page contains mods that greatly overhaul the game by making it much more challenging (and more rewarding in turn) and adding new features. These mods are not recommended for a first playthrough of the game. It is especially important to thoroughly read the descriptions of these mods, so you know what has changed and how to use the new features. Every mod in this section should be considered to be subjective and as such they may not be suitable for everyone.

JSawyer Ultimate Edition

Installation instructions:

  • Main File - JSawyer Ultimate Edition
  • Optional File - JSawyer Ultimate Edition - Push's Tweaks (select the name that matches the file on Nexus when the installation prompt comes up)
  • Additionally install JSUE - YUP Patch (select the name that matches the file on Nexus when the installation prompt comes up)
An updated and fixed version of the gameplay re-balance mod made by the lead director of the game, Josh Sawyer.
The INI for this mod contains many configurable features, such as an adjustable level cap and carry capacity formula. You should look through the INI and see if there are any features you would like to change. For example, you can set fCarryWeightBase to 150 if you want the vanilla carry weight formula back.

Mojave Arsenal

Installation instructions:

  • Main File - Mojave Arsenal
  • Optional Files - INI Config - Gun Runners' Arsenal Merged (select the name that matches the file on Nexus when the installation prompt comes up)
Adds ammo variants, reloading parts, and weapon mods as loot; provides comprehensive options for configuring and integrating Gun Runners' Arsenal; fixes item naming conventions; and improves recipes.

Famine

Installation instructions:

  • Main File - Famine
Dynamically lowers the amount of loot in the world to create a harsher and less forgiving economy.

Food Effect Tweaks

Installation instructions:

  • Main File - Food Effect Tweaks
Lowers the amount of healing provided by food items, making the distinction between the Survival and Medicine skills more important.

Player Combat Priority

Installation instructions:

  • Main File - Player Combat Priority
Makes enemies focus on the player in combat rather than solely companions.

Essential Vanilla Enhancements Merge

Installation instructions:

  • Main File - Vanilla Enhancements Merged
  • Optional File - Vanilla Enhancements Merged - JSawyer Ultimate Push's Tweaks Patch (select the name that matches the file on Nexus when the installation prompt comes up)
  • Additionally install Essential Vanilla Enhancements Merged - JSUE Patch and Essential Vanilla Enhancements Merged - Mojave Arsenal Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Compilation of many smaller gameplay-related mods, all updated, fixed, and optimized to work together.

Better Character Creation

Installation instructions:

  • Main File - Better Character Creation
Improves the character creation by speeding up the process, adding specialized gear based on your tag skills, and making Wild Wasteland an opt-in feature rather than a trait.
